Project 1 - Planets and Stars

Step 1: Idea

The projects primary purpose was to help us get an understanding of how to create a simple program on processing that would emulate a real life system. Eg - Sunrise and Sunset.

Step 2: Interaction diagram

The first step of the project was to create a diagram that represents the entire function and flow of all the elements that we will be using in the program. The task was to determine and define all the characteristics properties and functions and interactions between all the elements.

Step 3: Processing

After determining the interaction diagram, the next step was to make a form to the concept. The background and ground were images. The main star was a colored ball and the main design and concept was to get the shooter to work used on the click.

Step 4: Star Design

The star was also a smaller object which would take a random color. But it had 2 primary functions - to move in a random direction and interact with the planet when it collides.

Step 5: Implementation

Using the X coordinate, Velocity and Y Coordinate variables - the direction and speed of the star was determined. To make it more dynamic all these were random making it a much more interactive and fun system. The star just looped around from one side to the other making it emulate an apparent moment of the start across the horizon.
